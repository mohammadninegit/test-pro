دو نوع مخزن داریم

مخزن محلی          local repository
مخزن راه دور       remote repository


local repository 
همون مخزنی هست که روی سیستم خودمون اجرا کردیم


remote repository 
مخزنی هست که روی سرویس های گیت هاب یا گیت لب میسازیم
و غیر از خودم هر کسی میتونه بهش دسترسی داشته باشه

طبق توضیحات داده شده بعد از ایجاد مخزن محلی برای اتصال مخزن محلی به راه دور
از دستور پایین استفاده میکنیم

git remote add origin https://github.com/mohammadninegit/test-pro.git

اگر میخوایم دستوراتی رو که در مخزن محلی داریم رو منتقل کنیم به مخزن راه دور
و منتقل کنیم به گیت هاب از دستور زیر استفاده میکنیم

git push
